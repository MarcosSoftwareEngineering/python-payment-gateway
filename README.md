Este é um projeto desenvolvido como parte do meu aprendizado em **Engenharia de Software**, focado na criação de uma interface de processamento de pagamentos utilizando Python. O objetivo principal é simular o fluxo de transações financeiras de forma segura e eficiente.

## 🚀 Funcionalidades

- [x] Validação de dados de entrada.
- [x] Simulação de processamento de transações.
- [ ] Integração com APIs externas (Em desenvolvimento).
- [ ] Geração de logs de transação.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.x
- **Ambiente:** VS Code / Terminal Interativo

## 📂 Estrutura do Projeto

O projeto está organizado da seguinte forma:

* `main.py`: Ponto de entrada da aplicação onde a lógica principal é executada.
* `.gitignore`: Arquivo para ignorar pastas desnecessárias (como ambientes virtuais).
* `README.md`: Documentação oficial do projeto.
